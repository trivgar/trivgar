# Grit City Sim Racing — Brand Package Plan

## Overview

Full branding package for **Grit City Sim Racing**, a Tacoma-based sim racing community, and its flagship AC Evo server **Grit City Green Hell** (Nordschleife track day, slower cars).

## Brand Identity

- **Audience:** Late-80s gamers, Gran Turismo / retro racing generation
- **Vibe:** Gritty, retro, industrial — grassroots track day energy
- **Color Palette:** Industrial Tacoma
  - Dark charcoal (primary background)
  - Rust orange (accent / highlight)
  - Off-white / cream (text / contrast)
- **Themes:** Racing, Tacoma, worn/industrial textures

## Deliverables

### 1. Logo System
- **Primary logo** — Full mark incorporating racing + Tacoma elements, with "Grit City Sim Racing" text
- **Icon / avatar** — Simplified version for small spaces (Discord avatar, Steam profile, favicons)
- **Horizontal banner** — Wide format for headers and overlays
- **Badge / emblem** — Vintage racing decal style, good for merch and stickers

Each logo variant provided in:
- Full color (on dark background)
- Full color (on light background)
- Single color (white)
- Single color (dark)

### 2. Color Definitions
- Exact hex/RGB values for all brand colors
- Primary, secondary, and accent color usage guidelines
- Example do's and don'ts

### 3. Typography
- Primary font (headings, logo lockups)
- Secondary font (body text, UI)
- Font pairing rationale

### 4. Discord Assets
- Server icon (logo avatar version)
- Server banner
- Role icons (if applicable)
- Welcome/info channel header graphics

### 5. Social Media Templates
- Profile picture (universal)
- Banner/cover images (Discord, Twitter/X, YouTube)
- Event announcement template
- Session result / highlight template
