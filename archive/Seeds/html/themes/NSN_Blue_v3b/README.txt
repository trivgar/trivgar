1. Introduction and Requirements
--------------------------------
NSN_Blue_v3b is a PHP-Nuke Theme.
It requires PHP-Nuke 5.4.0 to 5.6.0 installed.
Dual Display has been fixed to display upto the set $storyhome limit,
it will also now work with categories.php and News/index.php without
having to modify these files. topics.php from NSN Topics will also
display the dual column display without modifing themeindex( to
themeindex2(.


2. Installation
---------------
Extract archive to temp dir and copy included as noted below:
-------------------------------------------------------------
README.txt (do not upload, instructions only)
themes/NSN_Blue_v3b/*.* ---------------> themes/NSN_Blue_v3b/*.*
themes/NSN_Blue_v3b/images/*.* --------> themes/NSN_Blue_v3b/images/*.*
themes/NSN_Blue_v3b/language/*.* ------> themes/NSN_Blue_v3b/language/*.*
themes/NSN_Blue_v3b/style/*.* ---------> themes/NSN_Blue_v3b/style/*.*


3. Use
------
After uploading all files as shown above you can select this theme in the "User"
interface and in the "Admin Settings" interface. The theme is best viewed at
1024x768 Hi Color display.
