This would be the first theme i've released to public, not so great but interesting =p
The default layout of this theme is the flash one. I didnt add sound effect to speed up loading time, theres a cdplayer that comes with the flash theme, its loaded from a movie clip so it doesnt effect loading time, and it has a lil preloader. The site title is very easy to config, here is how.

-Open sitename.txt-
just replace the "yoursite.com" to whatever u want and thats it.
Then just chmod the file to 777, dont really have to, but just incase lol.

-Layout change-
For those that rather use the non flash header, heres how.
Theres 2 non flash layouts, "header_non_flash" and "header_non_flash_wide" html. all u have to do is rename the header html file u want to header.html, and thats it.

Note: before u rename the header.html file, make sure u rename the default one to something else first. and thats it =) enjoy!
