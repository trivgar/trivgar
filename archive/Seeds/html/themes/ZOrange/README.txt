1. Introduction and Requirements
--------------------------------
ZOrange is a PHP-Nuke Theme.
It requires PHP-Nuke 5.3.0, 5.3.1, or 5.4.0 installed


2. Installation
---------------
Extract archive to temp dir and copy included as noted below:
-------------------------------------------------------------
themes/ZOrange/readme.txt (do not upload, instructions only)
themes/ZOrange/*.* ----------------> themes/ZOrange/*.*
themes/ZOrange/images/*.* ---------> themes/ZOrange/images/*.*
themes/ZOrange/language/*.* -------> themes/ZOrange/language/*.*
themes/ZOrange/style/*.* ----------> themes/ZOrange/style/*.*


3. Use
------
After uploading all files as shown above you can select this theme in the "User"
interface and in the "Admin Settings" interface. There is a variable to set the
table width ($thewidth) to your liking. You can use a percentage or a pixelular
width (i.e. 95% or 770)
