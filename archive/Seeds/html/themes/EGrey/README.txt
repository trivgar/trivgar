1. Introduction and Requirements
--------------------------------
EGrey is a PHP-Nuke Theme.
It requires PHP-Nuke 5.3.0, 5.3.1, or 5.4.0 installed


2. Installation
---------------
Extract archive to temp dir and copy included as noted below:
-------------------------------------------------------------
themes/EGrey/readme.txt (do not upload, instructions only)
themes/EGrey/*.* ------------------> themes/EGrey/*.*
themes/EGrey/images/*.* -----------> themes/EGrey/images/*.*
themes/EGrey/language/*.* ---------> themes/EGrey/language/*.*
themes/EGrey/style/*.* ------------> themes/EGrey/style/*.*


3. Use
------
After uploading all files as shown above you can select this theme in the "User"
interface and in the "Admin Settings" interface. There is a variable to set the
table width ($tableset) to your liking. If you set $tableset to 1 then the table
is 760 pixels wide and uses a 760x95 header graphic, if you set it to 2 then
the table width is 980 pixels wide and uses a 980x125 header graphic, and if you
set it to 3 then it uses a 1230x155 header graphic.
