What is Claude Code?

Claude Code is an AI assistant that works directly with your files and folders. It can read documents, help you write and edit, organize research, and much more.

  • Read and understand your documents
  • Help draft, edit, and refine writing
  • Organize and summarize research materials
  • Create outlines, syllabi, and course materials
  • Analyze data and generate reports
  • Answer questions about your files

It uses your Claude Pro subscription ($20/month) ‚Äî no separate API costs. When you first run it, it opens a browser window to sign into your Claude account.

Opening Claude Code (The Easy Way)

Step 1: Navigate to Your Folder in File Explorer

  1. Open File Explorer (press Win + E)
  2. Navigate to the folder containing your project or documents
  3. For example: Documents ‚Üí Research ‚Üí My Paper

Step 2: Right-Click ‚Üí Open Git Bash Here

  1. Once you're in your folder, right-click on empty space
  2. Click "Open Git Bash here"
  3. A terminal window will open, already in that folder
Tip: Hold Shift while right-clicking for additional options.

Step 3: Type claude and Press Enter

claude

The first time, it will open a browser window to sign into your Claude account. After that, it just starts up directly.

‚ö° Power-Ups for Claude Code

These optional add-ons extend what Claude Code can do. Install only what you need.

Enable Web/URL Reading

By default, Claude Code can't read web pages. Add this to let it fetch and read URLs:

Install the Fetch MCP Server

npm install -g @anthropic-ai/mcp-server-fetch

Add to Claude's Config

In Claude Code, run:

/mcp add fetch -- mcp-server-fetch

Or manually edit ~/.claude/config.json:

{
  "mcpServers": {
    "fetch": {
      "command": "mcp-server-fetch"
    }
  }
}

Now you can ask Claude to read web pages:

> read https://example.com/article and summarize the key points

Enable PDF Reading

To let Claude read PDF files directly, install poppler-utils:

On Windows (via Git Bash)

Download and install from: Poppler for Windows

Add the bin folder to your PATH.

Then Claude can extract text from PDFs:

> read the PDF in this folder and summarize section 3

What is Tailscale?

üîó The Simple Explanation

Tailscale is like a secret tunnel between your devices. It connects your phone, laptop, and Raspberry Pi together in a private network ‚Äî even when they're in completely different locations.

üåê Why This Matters for Your Pi-hole

Right now, your Pi-hole only blocks ads when you're at home. With Tailscale:

  • ‚úÖ Ad-blocking on your phone at coffee shops, airports, work
  • ‚úÖ Access Pi-hole dashboard from anywhere
  • ‚úÖ SSH into your Pi remotely for maintenance
  • ‚úÖ All traffic encrypted and private

üîí How It Works

Your Phone (anywhere) ‚Üê‚îÄ‚îÄüîê‚îÄ‚îÄ‚Üí Tailscale ‚Üê‚îÄ‚îÄüîê‚îÄ‚îÄ‚Üí Your Pi (at home)

When you enable your Pi as an "exit node," all your traffic goes through your Pi first ‚Äî ads get blocked before reaching your device.

üí∞ Cost: Free for personal use (up to 100 devices).

Part 1: Connect to Your Pi

Step 1.1: SSH into your Pi

From Git Bash on your PC:

ssh pihole@YOUR_PI_IP

Replace YOUR_PI_IP with your Pi's IP (e.g., 192.168.1.55)

Password: raspberry

Part 2: Install Tailscale on Your Pi

Step 2.1: Install Tailscale

curl -fsSL https://tailscale.com/install.sh | sh

Step 2.2: Start as Exit Node

sudo tailscale up --advertise-exit-node

Copy the URL it shows and open it in your browser to sign in.

Step 2.3: Approve Exit Node

  1. Go to Tailscale Admin Console
  2. Find your Pi ‚Üí click ‚ãÆ ‚Üí Edit route settings
  3. Enable "Use as exit node" ‚Üí Save

Step 2.4: Get Your Tailscale IP

tailscale ip -4

Write down this 100.x.x.x IP ‚Äî you'll use it to connect!

Part 3: Connect Your Devices

On Your Phone

  1. Download Tailscale from App Store / Play Store
  2. Sign in with the same account
  3. Tap ‚ãÆ ‚Üí Use exit node ‚Üí select your Pi

On Your Laptop

  1. Download from tailscale.com/download
  2. Sign in ‚Üí click tray icon ‚Üí Exit node ‚Üí select Pi

Part 4: OpenClaw ‚Äî Always-On AI (Optional)

OpenClaw is an AI gateway that runs on your Pi. Chat with AI via Telegram, Discord, or web ‚Äî accessible from anywhere via Tailscale.

Note: OpenClaw requires an API key from an AI provider. Choose one below based on your preference and budget.

Choose Your AI Provider

üü£ Anthropic (Claude) Pay as you go

The same Claude you use in Claude Code ‚Äî fast, capable, great for writing.

  1. Go to console.anthropic.com
  2. Sign up or log in
  3. Go to API Keys ‚Üí Create Key
  4. Copy the key (starts with sk-ant-)

Pricing: ~$3 per million input tokens, ~$15 per million output tokens (Sonnet)

üîµ Google (Gemini) Free tier available!

Google's AI ‚Äî generous free tier, good for experimentation.

  1. Go to aistudio.google.com/app/apikey
  2. Sign in with Google
  3. Click Create API Key
  4. Copy the key

Pricing: Free tier includes 15 requests/minute, 1 million tokens/month!

üü¢ OpenAI (ChatGPT/GPT-4) Pay as you go

The original ChatGPT ‚Äî widely known, lots of integrations.

  1. Go to platform.openai.com/api-keys

Part 5: Install OpenClaw

Step 5.1: Install Node.js

curl -fsSL https://deb.nodesource.com/setup_22.x | sudo bash -
sudo apt install -y nodejs

Verify: node --version should show v22.x.x

Step 5.2: Install OpenClaw

sudo npm install -g openclaw

Step 5.3: Initialize OpenClaw

openclaw init

It will ask for your API key ‚Äî paste the one you got from your chosen provider.

Step 5.4: Configure for Remote Access

nano ~/.openclaw/openclaw.yaml

Set the bind address:

gateway:
  port: 18789
  bind: tailnet

Save: Ctrl+O, Exit: Ctrl+X

Step 5.5: Start OpenClaw

openclaw gateway start

Access it at: http://YOUR_TAILSCALE_IP:18789

Part 6: Run OpenClaw on Startup (Optional)

Create a systemd Service

sudo nano /etc/systemd/system/openclaw.service

Paste:

[Unit]
Description=OpenClaw AI Gateway
After=network.target

[Service]
Type=simple
User=pihole
ExecStart=/usr/bin/openclaw gateway start --foreground
Restart=on-failure
RestartSec=10

[Install]
WantedBy=multi-user.target

Enable and start:

sudo systemctl daemon-reload
sudo systemctl enable openclaw
sudo systemctl start openclaw
