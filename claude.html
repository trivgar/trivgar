Claude Code - Getting Started

Your AI coding assistant in the terminal

What is Claude Code?

Claude Code is an AI assistant that lives in your terminal. It can:

  • Read and understand your code files
  • Write new code or edit existing files
  • Run terminal commands
  • Explain how things work
  • Debug problems and suggest fixes
  • Help you learn as you build

Think of it as having a senior developer sitting next to you, ready to help with whatever you're working on.

Opening Claude Code

Step 1: Open Git Bash

  1. Press Win key
  2. Type git bash
  3. Press Enter

Step 2: Navigate to Your Project

Use cd to change directories:

# Go to your Documents folder
cd ~/Documents

# Go to a specific project
cd ~/Documents/my-project

# Go up one folder
cd ..

# See what's in the current folder
ls
Tip: You can drag a folder from File Explorer into Git Bash to paste its path automatically!

Step 3: Start Claude

claude

That's it! Claude will start and show you a prompt where you can type.

Basic Usage

Just type what you want in plain English:

# Ask about your project
> what files are in this project?

# Get help with code
> explain how the login function works

# Make changes
> add a dark mode toggle to the settings page

# Fix bugs
> the save button isn't working, can you debug it?

# Learn something
> how do I make an API call in JavaScript?
Note: Claude can see all the files in your current folder and subfolders. It reads them to understand your project.

Useful Commands

Command What it does
/help Show all available commands
/clear Clear the conversation and start fresh
/compact Summarize the conversation to save memory
/cost See how much you've used this session
/quit or Ctrl+C Exit Claude

Tips for Better Results

Be Specific

Instead of:

> make it better

Try:

> refactor the handleSubmit function to use async/await instead of .then()

Give Context

> I'm building a recipe app. Add a feature to save favorite recipes to local storage.

Ask for Explanations

> explain what you changed and why

Review Before Accepting

Claude will show you what it wants to change before doing it. Read the diff and type y to accept or n to reject.

Important: Always review changes before accepting them. Claude is helpful but not perfect. You're the final decision maker.

Starting a New Project

# Create a new folder
mkdir my-new-project
cd my-new-project

# Start Claude
claude

# Ask it to help you set up
> help me create a basic React app with a todo list

Claude can scaffold entire projects, install dependencies, and get you started quickly.

Working on Existing Code

# Navigate to the project
cd ~/Documents/existing-project

# Start Claude
claude

# Let it learn about the project
> give me an overview of this codebase

# Then start working
> add input validation to the signup form

Keyboard Shortcuts

Shortcut Action
Ctrl + C Cancel current operation / Exit
↑ / ↓ Scroll through previous messages
Ctrl + L Clear the screen
Esc Cancel multi-line input

Troubleshooting

"claude" is not recognized

Close Git Bash completely and reopen it. If still not working:

npm install -g @anthropic-ai/claude-code

Claude seems stuck

Press Ctrl + C to cancel, then try again.

Too much output

Use /clear to start a fresh conversation.

Need help?

Just ask Claude: I'm having trouble with X, can you help?
