What is Claude Code?

Claude Code is an AI assistant that works directly with your files and folders. It can read documents, help you write and edit, organize research, and much more.

  • Read and understand your documents
  • Help draft, edit, and refine writing
  • Organize and summarize research materials
  • Create outlines, syllabi, and course materials
  • Analyze data and generate reports
  • Answer questions about your files

Think of it as having a capable research assistant who can see all your project files and help you work through them.

Opening Claude Code (The Easy Way)

Step 1: Navigate to Your Folder in File Explorer

  1. Open File Explorer (the folder icon in your taskbar, or press Win + E)
  2. Navigate to the folder containing your project or documents
  3. For example: Documents ‚Üí Research ‚Üí My Paper

Step 2: Right-Click ‚Üí Open Git Bash Here

  1. Once you're in your folder, right-click on empty space
  2. Click "Open Git Bash here"
  3. A terminal window will open, already in that folder
Tip: You can also hold Shift while right-clicking to see additional options if "Open Git Bash here" doesn't appear immediately.

Step 3: Type claude and Press Enter

claude

That's it! Claude will start and you can begin working.

Useful Commands

Command What it does
/help Show all available commands
/clear Clear the conversation and start fresh
/compact Summarize the conversation to save memory
/cost See how much you've used this session
/quit or Ctrl+C Exit Claude

Tips for Better Results

Be Specific About What You Want

Instead of:

> make this better

Try:

> make this introduction more engaging by starting with a concrete example 
  instead of a general statement

Give Context

> I'm writing a journal article for Technical Communication Quarterly. 
  Help me revise this methods section to match their style guidelines.

Ask for Explanations

> explain why you made those changes

Iterate and Refine

> that's good but make it less formal ‚Äî this is for undergraduate students

> try again but keep the original opening sentence

Review Before Accepting Changes

When Claude wants to edit a file, it will show you exactly what it plans to change. Review the changes and type y to accept or n to reject.

Important: Always review changes before accepting them. Claude is helpful but not perfect ‚Äî you're the expert on your own work.

Troubleshooting

"claude" is not recognized

Close Git Bash completely and reopen it. If still not working, reinstall:

npm install -g @anthropic-ai/claude-code

Claude seems stuck or slow

Press Ctrl + C to cancel, then try again with a simpler request.

Too much conversation history

Use /clear to start fresh, or /compact to summarize and continue.

Claude made changes I don't want

If you accepted changes you didn't mean to, use Ctrl + Z in your text editor to undo, or check your file's version history.

Need help?

Just ask Claude: I'm having trouble with X, can you help?
